# Sales Performance Dashboard using Power BI

## Project Overview

This project showcases an interactive Sales Performance Dashboard developed using Microsoft Power BI. The dashboard provides valuable insights into sales performance, profitability, customer behavior, product performance, and regional trends. It enables users to monitor key business metrics through interactive visualizations and dynamic filters, supporting better business decision-making.

---

## Objectives

- Analyze overall sales and profit performance.
- Identify top-performing products and customer segments.
- Compare sales and profit across different regions.
- Study the impact of discounts on profitability.
- Build an interactive dashboard for business analysis.

---

## Tools & Technologies Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- CSV Dataset

---

## Dataset Information

The dashboard is built using the Superstore Sales dataset containing retail sales transactions.

The dataset includes:

- Order Details
- Customer Information
- Product Information
- Category & Sub-Category
- Region & State
- Sales
- Profit
- Discount
- Quantity
- Shipping Details

---

## Dashboard KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Profit Margin

## Dashboard Visualizations

- Monthly Sales Trend (Line Chart)
- Sales by Category (Column Chart)
- Profit by Region (Bar Chart)
- Sales by Segment (Donut Chart)
- State-wise Sales Distribution (Map)
- Top 10 Selling Products (Bar Chart)
- Discount vs Profit Analysis (Scatter Plot)

---

## Filters (Slicers)

- Year
- Region
- Category
- Segment

---

## Key Business Insights

- Technology category generated the highest sales revenue.
- The West region contributed significantly to overall sales and profit.
- Higher discounts generally resulted in lower profitability.
- A small number of products generated a large share of total sales.
- Interactive filters allow users to analyze sales performance from multiple perspectives.
